# Tmux

# prefixed by CTRL-A (Default is CTRL-B)
- c - New window
- n - next
- p - preview
- % - split vertically
- " - split horizontally
- <arrow key> - navigate beteween splits
- space - reorganize splits
- x - close window
- d - detach session
- w - show list of windows so it can be selected
- , - rename the current window
- f <name> - go to the named window
